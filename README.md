# Auto Video (Astro)

English multi-page Astro website for an AI-powered auto video product.

## Features
- No-login experience
- Mock prompt-to-video generation flow
- Demo gallery pages
- Customer testimonials
- Contact page sending to `admin@autovideo.videi`
- Cloudflare Pages-ready deployment as a static Astro site

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
